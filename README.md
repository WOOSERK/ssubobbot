# 슈밥봇 - 숭실대학교 학식 정보를 알려주는 카카오톡 챗봇

### 사용 기술
카카오 i 오픈빌더(카카오톡 채널 챗봇 API), 파이썬 3.7.9, 라이브러리 scrapy, AWS lambda, dynamoDB, AWS EC2(cron)
